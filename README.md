## Solve Problem install packege maatwebsiteexcel

-   open file php.ini
    before
    `extension=gd;`  
    after
    `extension=gd`
-   execute the command to change your cache

```json
composer require psr/simple-cache ^2.0
```

-   Then install maatwebsiteexcel

```json
    composer require maatwebsite/excel
```
