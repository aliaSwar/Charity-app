## About Charity App

عبارة عن نظام أتمتة لجمعية إنعاش الفقير الخيرية يتضمن :

-   قسم الإدراج .
-   قسم الإعانات.
-   قسم الكفالات.
-   قسم التقارير المالية.
-   قسم الموظفين.
-   قسم تواصل الكفلاء مع الجمعية إدارة بياناتهم.

## التقنيات المستخدمة

-   laravel in backend.
-   html css js in front end web app.
-   android java in front ent app.
